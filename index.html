Session overview

This two-hour hands-on workshop introduces vibe coding to executive master students with no engineering background. You steer; the AI writes. The goal is to ship a working prototype of your own idea before the session ends.

A few things to set up beforehand so you can spend the session building, not configuring. Follow them in order:

1
Required

A Claude account with an active Pro subscription

You'll use Claude throughout the hands-on exercises. A Pro plan (or higher) is needed for Artifacts and Claude Design usage limits. claude.ai

2
Required

A GitHub account, and basic familiarity with it

You'll use GitHub during the practice session. Create an account and get comfortable with the basics beforehand. Sign up ↗ · Introduction to GitHub ↗ · Know more: GitHub 101 ↗

3
Required

One idea to prototype

Come with a project or idea you'd like to build, anything goes. If you don't have one, we'll provide a default case, but you'll get far more out of the workshop (and reflect more deeply) by bringing your own.

4
Recommended

Claude desktop app installed

Smoother experience than the browser. Install Claude for macOS or Windows ↗
